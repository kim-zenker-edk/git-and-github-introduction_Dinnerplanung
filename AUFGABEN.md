# Aufgaben 

## Erste Aufgabe: Rezepte hinzufügen
1. Such dir einen Gang aus für den du eine Rezeptidee hast
2. Erstelle einen neuen branch um das Rezept hinzuzufügen
    - der branch sollte feature/<name>-<gang> heißen, also zb feature/kim-vorspeise
    - über die commandline dann zb: 
        git checkout -b feature/kim-vorspeise
3. Füge dein Rezept hinzu. Nutze hierfür das template 
4. stage deine Änderung mit git add 
5. committe deine änderungen mit git commit - und denk daran eine commit message hinzuzufügen
6. veröffentliche deine änderungen mit git push (es kann sein dass das nicht sofort klappt - lies die Fehlermeldungen, um herauszufinden warum)


## Zweite Aufgabe: Pull Requests, Merges
Wir erstellen gemeinsam pullrequests für die branches 

## Dritte Aufgabe: merge Konflikte lösen
Füge für deinen ursprünglichen Gang alle Zutaten auf die Einkaufsliste hinzu.
